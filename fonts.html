Fonts

There are two different types of fonts used:

  • Web safe fonts
  • Web fonts.
    •

Web safe fonts are the fonts that are pre-installed in the operating system. Web fonts are the fonts that are not pre-installed in the operating system. Web fonts are downloaded by the browser when needed.

Web safe fonts

Some of the examples of web safe fonts are:

  • Arial
  • Verdana
  • Times New Roman
  • Georgia
  • Courier New
  • Lucida Console

Web fonts

Some of the examples of web fonts are:

  • Roboto
  • Big Shoulder Stencil
  • Emblema One
  • Bebas Neue
